## Features

- ⚡️ Vue 3, Vite 2, Vuex 4, Tailwindcss 2
- 🚀 Steps routed system

## Getting Started

### npm:
```sh
npm install
npm run dev
npm run build
npm run serve
```
### yarn:
```sh
yarn
yarn dev
yarn build
yarn serve
```
