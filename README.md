# Formyle

Advanced Vue 3 form conception

## Features

- ⚡️ Vue 3, Vite 2, Vuex 4, Tailwindcss 2
- 🚀 Steps routed system
- 🖨 Multiple versions (A/B Testing, source, ...)
- Soon (...)

## Getting Started

### npm:
```sh
npm install
npm run dev
npm run build
```
### yarn:
```sh
yarn
yarn dev
yarn build
```
